# Портфолио

Простое React-приложение с портфолио проектов с фильтрами.

## Функции

- Компонент Portfolio с управлением состоянием фильтров
- Toolbar для фильтрации по категориям (All, Websites, Flayers, Business Cards)
- ProjectList с отображением проектов в masonry layout
- Активный фильтр визуально выделен

## Демо
Посмотреть демо: [GitHub Pages](https://marl3ndev.github.io/portfolio/)

## Начало работы

1. Установите зависимости: `yarn install`
2. Запустите приложение: `yarn dev`
3. Откройте http://localhost:5173 в браузере

## Сборка

Запустите `npm run build` для сборки приложения в продакшн..
